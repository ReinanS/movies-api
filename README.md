## Installation

#### Requirements
<ul>
  <li>Gradle</li>
  <li>JDK 11</li>
  <li>An IDE of your preference</li>
</ul>

Clone the repository

```
git clone https://github.com/ReinanS/movies-api.git
```
<br/>
Open up your IDE and import the project as a Maven project.
<br/><br/>
Download Dependencies
